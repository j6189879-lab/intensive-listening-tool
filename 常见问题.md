# 常见问题解决

## WebSocket 连接失败

### 1. 检查 API 配置

**问题：** APPID、APIKey、APISecret 填写错误

**解决：**
- 登录讯飞开放平台：https://www.xfyun.cn/
- 进入控制台 → 我的应用
- 确认复制的是**语音听写（流式版）**的配置
- 注意不要有多余的空格

### 2. 检查应用状态

**问题：** 应用未开通或已过期

**解决：**
- 进入控制台 → 我的应用
- 确认应用状态为"正常"
- 查看剩余调用次数
- 如果是新应用，等待 5-10 分钟生效

### 3. 检查网络连接

**问题：** 无法连接到讯飞服务器

**解决：**
- 确认网络正常
- 尝试访问 https://www.xfyun.cn/ 测试连接
- 检查防火墙是否拦截 WebSocket 连接

### 4. 查看详细错误

**步骤：**
1. 按 F12 打开浏览器开发者工具
2. 切换到"控制台"标签
3. 点击"查看答案"触发识别
4. 查看红色错误信息

**常见错误码：**
- `10105`: APPID 不存在或格式错误
- `10106`: APIKey 不存在或格式错误
- `10107`: 签名验证失败（APISecret 错误）
- `10110`: 无权限访问该服务
- `10163`: 请求的 APPID 与 APIKey 不匹配
- `10200`: 读取数据超时
- `10222`: 应用已过期

## 识别结果不准确

### 1. 音频质量问题

**解决：**
- 使用清晰的音频文件
- 避免背景噪音过大
- 确保音频不是压缩过度的低质量文件

### 2. 语言设置

**当前设置：** 英文识别 (`language: 'en_us'`)

**如果需要识别中文：**
修改 `xunfei-asr.js` 第 52 行：
```javascript
language: 'zh_cn',  // 改为中文
```

### 3. 句子太短或太长

**建议：**
- 单句时长：1-15 秒
- 太短可能识别不到
- 太长可能超时

## 翻译失败

### 1. MyMemory API 限流

**问题：** 每天 5000 字符限制

**解决：**
- 等待第二天重置
- 或手动添加翻译

### 2. 网络问题

**解决：**
- 检查网络连接
- 稍后重试

## 其他问题

### 音频无法播放

**解决：**
- 确认浏览器支持该音频格式
- 推荐使用 Chrome 或 Edge 浏览器
- 尝试转换音频格式（MP3/WAV）

### 数据丢失

**说明：**
- 所有数据保存在浏览器 localStorage
- 清除浏览器数据会丢失
- 建议定期导出文本备份

### 性能问题

**如果音频很大（>10分钟）：**
- 分句可能较慢
- 建议分段处理
- 或使用较短的音频文件
