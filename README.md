# 精听训练工具

完全免费的英语精听训练工具，支持自动识别音频内容。

## 核心功能

✅ **自动音频分句** - 基于静音检测，无需手动分割
✅ **自动识别英文** - 使用 Groq Whisper API（完全免费）
✅ **按需翻译** - 只在查看答案时才翻译，节省额度
✅ **逐句播放** - 自动切分后逐句播放
✅ **播放控制** - 上一句/下一句/播放/暂停/速度调节
✅ **进度管理** - 自动保存学习进度
✅ **简洁 UI** - iPhone 风格设计

## 使用步骤

### 1. 获取免费 API Key

访问 [console.groq.com](https://console.groq.com)
- 免费注册账号（支持 Google 登录）
- 左侧菜单 → API Keys → Create API Key
- 复制密钥（格式：`gsk_...`）

### 2. 开始使用

1. 双击打开 `index.html`
2. 上传音频文件（MP3/WAV/M4A）
3. 等待自动分析和分句
4. 点击"自动识别文本"
5. 粘贴 Groq API Key
6. 等待识别完成（只识别英文，不翻译）
7. 开始精听训练

### 3. 精听流程

1. 点击播放按钮听当前句子
2. 多听几遍，尝试理解内容
3. 点击答案区域查看英文原文
4. **首次查看时会自动翻译成中文**（按需翻译，节省额度）
5. 使用上一句/下一句切换
6. 调节播放速度（0.5x - 2.0x）

## 费用说明

**完全免费！**

- **Groq Whisper API**：免费注册即用，每天约 28,800 秒音频额度
- **MyMemory 翻译 API**：免费，每天 5000 字符
- **按需翻译**：只翻译你查看的句子，大幅节省额度

## 手动添加文本（可选）

如果不想使用 API，也可以手动添加：

1. 点击"手动添加文本"按钮
2. 每行格式：`英文句子|中文翻译`
3. 保存即可

## 技术特点

- 纯前端应用，无需安装
- 数据保存在本地浏览器
- 支持离线使用（识别后）
- 翻译结果自动缓存
