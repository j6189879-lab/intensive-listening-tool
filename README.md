# 精听训练工具

一个完全免费的英语精听训练工具。上传音频自动分句，逐句反复听写，支持语音识别和翻译。

## 功能

- **🎵 音频上传与自动分句** — 上传 MP3 / WAV / M4A，基于静音检测自动切成句子片段
- **▶️ 逐句精听** — 上一句 / 下一句 / 播放 / 暂停，任意句子反复听
- **⏩ 变速播放** — 6 档速度调节（0.5x ~ 2.0x），慢速听清每个词
- **🎤 语音识别** — 浏览器内置 Web Speech API，免费识别英文语音（需 Chrome / Edge）
- **🌐 自动翻译** — 英文识别结果自动翻译为中文（MyMemory 免费 API）
- **📝 手动输入** — 也可手动粘贴文本（每行 `英文|中文翻译` 格式）
- **💾 本地存储** — 所有数据保存在浏览器，不上传服务器

## 使用方法

1. 用 **Chrome** 或 **Edge** 浏览器打开 `index.html`
2. 上传一个英语音频文件
3. 等待自动分句完成
4. 使用播放控制逐句练习
5. 点击答案区域查看语音识别结果和中文翻译
6. 也可点击「自动识别」批量识别所有句子

无需安装，无需注册，无需 API Key。

## 已知限制

- **语音识别**：通过浏览器麦克风捕获音频播放进行识别，非直接处理音频文件，准确率受环境和音频质量影响
- **翻译**：MyMemory API 免费层每日限制 5000 字符，超出后翻译可能为空
- **存储**：localStorage 通常限制 5-10 MB，超长音频的文本数据可能超出限制

## 其他转录方案

如果浏览器语音识别效果不理想，可先用以下工具获取文本，再手动粘贴到工具中练习：

- **剪映** — 自动识别字幕并导出
- **网易见外工作台** — 免费语音转写
- **讯飞听见** — 专业语音转写（付费）
