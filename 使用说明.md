# 精听训练工具 - 使用说明

## 快速开始（推荐方式）

由于国内网络环境限制，推荐使用以下方式：

### 方法1：使用剪映自动识别（最简单）

1. **用剪映转录音频**
   - 打开剪映专业版（免费）
   - 导入音频文件
   - 点击"文本" → "识别字幕" → "开始识别"
   - 等待识别完成（免费且准确）

2. **导出文本**
   - 识别完成后，点击"导出字幕"
   - 选择 SRT 或 TXT 格式
   - 保存到本地

3. **导入到精听工具**
   - 打开本工具，上传同一个音频文件
   - 等待自动分句完成
   - 点击"手动添加文本"
   - 将剪映导出的文本粘贴进去
   - 格式：每行一个句子，`英文|中文翻译`

### 方法2：使用网易见外工作台

1. 访问 https://jianwai.youdao.com/
2. 上传音频文件
3. 选择"语音转写"
4. 等待识别完成（免费，每天2小时）
5. 下载识别结果
6. 导入到本工具

### 方法3：使用讯飞听见

1. 访问 https://www.iflyrec.com/
2. 上传音频文件
3. 等待识别完成（新用户有免费时长）
4. 下载识别结果
5. 导入到本工具

## 文本格式说明

导入文本时，每行格式为：

```
英文句子|中文翻译
```

示例：

```
Hello, how are you today?|你好，你今天怎么样？
I'm doing great, thank you.|我很好，谢谢你。
What are your plans for the weekend?|你周末有什么计划？
```

## 翻译获取方式

如果只有英文，没有中文翻译：

1. **使用 DeepL**（推荐）
   - 访问 https://www.deepl.com/translator
   - 粘贴英文，自动翻译成中文
   - 复制翻译结果

2. **使用百度翻译**
   - 访问 https://fanyi.baidu.com/
   - 批量翻译

3. **使用本工具的按需翻译**
   - 导入只有英文的文本（格式：`英文|`）
   - 精听时点击查看答案
   - 工具会自动调用免费翻译 API

## 完整工作流程

1. **准备音频** → 上传到剪映/网易见外
2. **自动识别** → 获得英文文本
3. **翻译文本** → 使用 DeepL 或百度翻译
4. **导入工具** → 粘贴到"手动添加文本"
5. **开始精听** → 逐句播放，查看答案

## 为什么不用 API？

- 国外 API（OpenAI、Groq）在国内访问不稳定
- 国内 API（讯飞、百度）需要复杂的签名认证，纯前端难以实现
- 使用现成工具更简单、更稳定、更快速
